About Page

Welcome to my Django Blog Application! This platform was developed with the purpose of connecting users through posts and interactions. Below is an overview of what you can do on this application:

  • User Registration and Profiles: Create an account and set your profile picture.
  • Posting and Viewing: Share your thoughts and see what others have posted, with an efficient pagination system to manage large amounts of content.
  • Quote of the Day: Enjoy a new inspirational quote every day.
  • File Upload to AWS S3: Securely upload and store your files on AWS S3.

This blog application is built using Django, a powerful web framework that enables rapid development and clean, pragmatic design. Here are some key aspects of the development process:

  • Manipulated Django settings to customize the application environment.
  • Created and managed models and views to handle the core functionality of the app.
  • Implemented pagination to manage and display large datasets effectively.
  • Integrated AWS S3 for storing and retrieving application files.
  • Deployed the application on Heroku, ensuring smooth and scalable deployment.

This project was inspired by a tutorial by Corey Schafer. You can watch the original tutorial here. I've added additional functionalities to have fun. I hope you enjoy using this application as much as I enjoyed building it!
